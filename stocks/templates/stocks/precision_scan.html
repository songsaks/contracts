5
Price Pattern — แท่งเทียนบอกอะไร
▲ Bullish Patterns — เพิ่ม BUY Score
▲ Bullish Engulf +10 แท่งขาขึ้นกลืนแท่งขาลง → แรงซื้อพลิกกลับ
▲ Hammer +10 ไส้ล่างยาว → ราคา reject แรงขาลงออก
▲ Morning Star +8 3 แท่ง reversal หลังดิ่ง
▲ Inside Bar↑ +6 breakout จาก consolidation
💡 Bullish pattern ที่ Entry Zone = สัญญาณ timing เข้าแม่น — น้ำหนักเพิ่ม BUY Score สูงสุด +10
▼ Bearish Patterns — หักออกจาก BUY Score
▼ Bearish Engulf −10 แท่งขาลงกลืนแท่งขาขึ้น → แรงขายแรง
▼ Shooting Star −8 ไส้บนยาว → ราคา reject แรงขาขึ้นออก
▼ Doji −5 แท่งลังเล body เล็กมาก → อ่อนแรง
— (ไม่มี pattern) 0 ไม่มีผลต่อ score
⚠️ ถ้าเห็น Bearish Pattern ที่ Zone — รอแท่งถัดไปก่อน อย่าเพิ่งรีบเข้า
6
REL MOM — หุ้นชนะตลาด (SET) หรือเปล่า?
Relative Momentum คือ return ของหุ้น เทียบกับ SET Index ช่วง 3 เดือน
ถ้าหุ้นวิ่งชนะตลาด → momentum แข็งแกร่ง → โอกาสถือต่อสูง (ระยะ 1–6 เดือน)
+15% ขึ้นไป → ชนะ SET แรงมาก — BUY +12 pts
+8% ถึง +15% → ชนะ SET ชัดเจน — BUY +9 pts
+3% ถึง +8% → ชนะ SET พอรับได้ — BUY +6 pts
0% ถึง +3% → เสมอ SET — BUY +3 pts
−5% ถึง 0% → แพ้ SET เล็กน้อย — เป็นกลาง
ต่ำกว่า −5% → แพ้ SET ชัดเจน — BUY −8 pts ⚠️
— → ไม่มีข้อมูล SET วันนั้น
💡 ระยะ 6 เดือน: เลือกหุ้นที่ REL MOM เป็นบวก จะช่วยให้อยู่ในหุ้นที่ momentum ยังแรงอยู่ ไม่ใช่หุ้นที่ตลาดทิ้งไปแล้ว

Trend Following (v4) — ยืนยันว่า Momentum มี Structure รองรับ
เหตุผล: Momentum อาจเป็นแค่การดีดตัวชั่วคราว — Trend Following ช่วยกรองเฉพาะหุ้นที่ โครงสร้างราคา ยืนยันว่า uptrend ยังอยู่ เหมาะสำหรับถือ 1-6 เดือน
EMA20 Slope EMA↑ +5 pts
slope = (EMA20today - EMA205d ago) / EMA205d ago x 100
slope > 0.1% = EMA20 กำลังขึ้น
slope ≤ 0.1% = EMA แบน/ลง
EMA20 ขึ้น = ราคาเฉลี่ย 20 วันมีทิศทาง ไม่ใช่ sideways
HH/HL Structure HH/HL +4 pts
Higher High + Higher Low ใน 20 แท่งล่าสุด
Swing High = สูงกว่า 2 แท่งซ้าย+ขวา
Swing Low = ต่ำกว่า 2 แท่งซ้าย+ขวา
HH & HL ครบ = uptrend structure
Dow Theory: ขาขึ้นที่แท้ต้องมี HH และ HL เรื่อยๆ
Trend Confirmed TREND✓ +10 pts
ต้องมี ทั้ง EMA↑ และ HH/HL
ทั้งคู่ = TREND✓ (+10 pts)
EMA↑ เท่านั้น = +5 pts
HH/HL เท่านั้น = +4 pts
TREND✓ = momentum + structure = ถือได้ยาว 1-6 เดือน
วิธีอ่าน Badge ใน SIGNAL column
TREND✓ EMA20 ขึ้น + HH/HL ครบ +10 pts

HH/HL โครงสร้างขาขึ้น (EMA แบน) +4 pts
