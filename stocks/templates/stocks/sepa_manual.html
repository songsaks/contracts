Filters — วิธีใช้แต่ละ Toggle

เปิด/ปิด Filter ตาม Market Condition

🎯 VCP Only

กรองเฉพาะหุ้นที่มี Volatility Contraction Pattern ชัดเจน

เปิดเมื่อ: ตลาดปกติ ต้องการความแม่นยำสูง
ปิดเมื่อ: ตลาดแรง / ต้องการดู Early Candidates ด้วย
📊 EPS/Rev ≥ 25% MINERVINI

กรองเฉพาะที่ผ่านเกณฑ์ Earnings ของ Minervini

เปิดเมื่อ: ต้องการ Quality สูงสุด / ตลาดยาก
ปิดเมื่อ: ดู Pipeline กว้างๆ หรือข้อมูล Earnings ยังไม่อัพเดท
🚫 ซ่อน AT/NEAR TP

ซ่อนหุ้นที่ราคาชนเป้าแล้ว (Upside < 5-10%)

เปิด (Default): กรองหุ้นที่ชนเป้าออก
ปิดเมื่อ: ต้องการดูทุกหุ้น รวมที่ชนเป้าแล้ว
Best Practice: ในตลาดปกติ เปิดทั้ง VCP Only + EPS/Rev ≥ 25% + ซ่อน AT/NEAR TP จะได้หุ้นที่ผ่านเกณฑ์ Minervini แบบครบ ≤ 5-10 ตัว — โฟกัสได้ง่าย

FAQ — คำถามที่พบบ่อย

ตอบทุกข้อสงสัยเกี่ยวกับ SEPA System

Q: ทำไมต้องรัน Precision ก่อน SEPA SET?
A: SEPA SET ดึงข้อมูลจาก PrecisionScanCandidate — ถ้าไม่มี Precision data ตาราง SEPA จะว่างเปล่า สำหรับ SEPA US แยก database ต่างหากโดยสมบูรณ์
Q: EPS Growth ใน SET แม่นไหม?
A: ข้อมูลมาจาก Yahoo Finance (earningsQuarterlyGrowth) ซึ่งอาจ lag 1-2 quarter — ควรยืนยันกับ Opportunity Day หรือ SET Portal ก่อนเข้าซื้อ SET แสดงเฉพาะ eps_growth ที่มีใน PrecisionScanCandidate
Q: ⚡ ACCEL tag แปลว่าอะไร?
A: EPS Acceleration — Forward EPS estimate สูงกว่า Trailing EPS หมายความว่านักวิเคราะห์คาดว่ากำไรจะเร่งขึ้นในอนาคต ซึ่ง Minervini ถือว่าสำคัญมาก (+10 bonus score)
Q: SEPA Score กี่คะแนนถึงซื้อได้?
A: แนะนำ ≥ 100 (Strong) ก็เข้าได้ถ้า VCP ชัดเจน แต่ที่ดีที่สุดคือ ≥ 140 (Elite) ที่ผ่านทุกเกณฑ์รวมถึง Earnings
Q: ใช้ SEPA SET กับ SEPA US ต่างกันอย่างไร?
A: SEPA SET — กรองจาก Precision Scan (ต้องรัน Precision ก่อน) · SEPA US — มี database แยกต่างหาก รัน scan เองได้อิสระ และมีข้อมูล ROE เพิ่มเติม เพราะ Yahoo Finance มีข้อมูล US ครบกว่า
Q: Scan บ่อยแค่ไหนดี?
A: สำหรับ SET รัน 2-3 ครั้ง/สัปดาห์ (หลัง Market close) · สำหรับ US รัน 1-2 ครั้ง/สัปดาห์ เพราะสแกน ~500 หุ้น ใช้เวลา 5-10 นาที
